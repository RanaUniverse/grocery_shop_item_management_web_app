# 🛒 Item Inventory Management System

I am making the **Inventory Management System** designed for managing grocery **shop items, stock levels, and basic product data**.

## 📌 Project Goal
This project aims to help small grocery shop owners:
- Store and manage item data
- Track stock levels
- Maintain an organized inventory system
- Reduce manual record-keeping


### Installation

How i started this repo?

```bash
uv init . --python 3.14.1
```
HOw i will start this web for now?
```bash
uv run main.py
```
